## Description
"모잇"은 기숙사생활하는 대학생을 위한 단건 배달 플랫폼입니다. 
앱에서 +버튼을 탭해 주문을 하면,
게시판에 글이 올라가게되고 참가 인원이 모이면 자동으로 주문이 되는 시스템입니다.

### 개발환경 

XCODE 13.1
iOS 14.0 이상 버전

### 외부라이브러리

'AlertToast'
'Alamofire', '~> 5.4'
